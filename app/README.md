### How to install
1) Clone the repository
2) Open in Android Studio
3) Paste your Yandex MapKit api key into ``local.properties`` file like this:
```
# other code
MAPKIT_API_KEY = 2ccdc51a-d356-7459-as8c-t432023b785a #your api key here
```
4) Build and run
